ERP Assistant App

Target audience — Child

Nowadays, sledding is still in the first phase of "child play development”. This project goal is to make sleding fit the last stage – game with rules.

Turn on the helmet

To activate the helmet, children should fasten the belt.

Connect all players’ helmet

System will randomly choose one of the player to be “IT”, and ture his/her helmet purple. The one chosen to be “IT”, will have special way to win the game.

After the IT is chosen,IT’s screen will be locked down for 10 seconds before start sledding. The rest of players have to start sledding ASAP before IT gets them.

Player IT

If the distance between IT and regular player within 1 meter, IT will get coins from the player.

Regular player

Regular players can earn coins by bypassing other players. They will earn more coins if they bypass IT.
